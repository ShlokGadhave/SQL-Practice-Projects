# SQL-Practice-Projects

This repository contains my SQL practice work performed on multiple datasets.
The goal of this project is to strengthen SQL skills used in real-world data analysis.

## 📊 What This Repository Includes

* SQL queries written using different datasets
* Practice of SQL concepts from **basic to advanced**
* Data analysis tasks such as filtering, aggregation, joins, and subqueries

## 🧠 SQL Concepts Used

* SELECT statements
* WHERE, ORDER BY, GROUP BY
* Aggregate functions (SUM, AVG, COUNT, MIN, MAX)
* JOINS (INNER JOIN, LEFT JOIN)
* Subqueries
* CASE statements
* Window functions
* Data filtering and transformation

## 📁 Dataset Sources

Public datasets used for practice such as movie data, sales data, and other sample datasets.

## 🎯 Purpose

To build strong SQL query writing skills and demonstrate practical data analysis using SQL.

## 🛠 Tools Used

* MySQL
* VS Code
